# 冯翰铭的个人博客

这是一个纯静态个人主页，聚焦嵌入式开发、无刷电机 FOC、PCB 设计和工程项目实践。

## 本地预览

直接打开 `index.html` 即可。

## 站点结构

- `index.html`：页面结构与文案
- `styles.css`：视觉样式与响应式布局
- `script.js`：滚动动效、导航高亮与时钟

## 发布说明

该站点适合直接部署到 GitHub Pages。

仓库建议：

- 个人主页：`52debugder.github.io`
- 普通项目页：任意仓库名

## 隐私说明

仓库默认不包含以下本地材料：

- 简历原始 `docx/pdf`
- 成绩单、证书扫描件等敏感图片

如需公开这些文件，请先自行脱敏后再添加到仓库。
